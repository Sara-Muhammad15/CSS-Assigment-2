Guides

HTML, CSS, and JS guides with examples and explanations.

Define terms with HTML

The <dfn> element defines a term. Example: 

<p>The term <dfn>API</dfn> refers to...</p>

Use data attributes

Data attributes (data-*) store custom data inside elements: 

<div data-user="123">Hello</div>
console.log(document.querySelector('div').dataset.user); // 123

Use cross-origin images

Cross-origin images require proper CORS headers and the crossorigin attribute: 

<img src="https://example.com/image.jpg" crossorigin="anonymous">

Add a hitmap on top of an image

Define clickable areas inside an image using <map> and <area>: 


<img src="example.jpg" usemap="#map">
<map name="map">
  <area shape="rect" coords="34,44,270,350" href="#section1">
</map>

Author fast-loading HTML pages

Minimize CSS/JS, optimize images, and use semantic tags for performance.

Add JavaScript

Add dynamic behavior to your page with JS: 

<button onclick="alert('Hello!')">Click Me</button>
